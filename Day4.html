Embedded System

The session was taken by Miss Amrita Joshi.


An embedded system is designed to run the electro-mechanical system with the help of computer software.
The three main components of embedded system are:
  • Sensor/ button (input)
  • Processor/ controller and
  • Actuator/LCD display/ motor driver (output)
